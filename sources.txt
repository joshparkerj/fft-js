This is a nodejs implementation of the Cooley-Tukey Fast Fourier Transform as described at https://en.wikipedia.org/wiki/Cooley%E2%80%93Tukey_FFT_algorithm.

I have included a test data set of the pageviews of the Utah Valley University Wikipedia page from https://tools.wmflabs.org/pageviews/?project=en.wikipedia.org&platform=all-access&agent=user&start=2015-07-01&end=2018-11-06&pages=Utah_Valley_University
